# Dhanalakshmi Painting Contractor — Website

This is a React + Vite + Tailwind project scaffold for a modern, responsive painting business website.

Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm start
```

Build

```bash
npm run build
```

Notes

- Update contact details, phone, and Google Maps embed in the `src/pages/Contact.jsx` file.
- Add real project images to `src/data/projects.js`.
